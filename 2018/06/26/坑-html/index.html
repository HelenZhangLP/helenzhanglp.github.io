web 页面生命周期

DOMContentLoaded

纯 html 被完全加载解析时，DOM 加载完毕，样式表、图片等其它资源可以未加载时，触发 DOMContentLoaded

load

浏览器加载了所有的资源，包括图像、样式表等

beforeunload/unload

离开当前页面时触发

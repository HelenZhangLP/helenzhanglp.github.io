Network-localhost/127.0.0.1/0.0.0.0的区别联系

魔女
2022-10-21
  • Network
名称 定义 特性
localhost 回路网络（loopback）的一个标准的主机名,对应 ip（127.0.0.1）ipv4/(::1)ipv6
127.0.0.1 or ::1 是一个回送地址，主机IP堆栈内部的IP地址
0.0.0.0 or :: 一个不可路由的元地址，用于指定无效、未知、或不适用的目标

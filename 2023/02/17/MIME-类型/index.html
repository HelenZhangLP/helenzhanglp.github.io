关于 MIME 类型

多用途互联网邮件扩展类型（Multipurpose Internet Mail Extensions）媒体类型是一种标准，用来表示文档、文件或字节流性质和格式。
type/subType

类型 描述 典型类型
text 普通文本，人类可读 text/plain 纯文本文档;
text/html html文档;
text/css css 文件;
text/javascript JavaScript 文件
image 某种图像，包含动态图，如 gif。使用 image 类型，但不包括视频 image/gif gif 图片
image/png png 图片
image/jpeg jpeg 图片
image/bmp, image/webp, image/x-icon, image/vnd.microsoft.icon
audio 某种音视文件 audio/midi, audio/mpeg, audio/webm, audio/ogg, audio/wav
video 某种视频文件 video/webm, video/ogg
application 某种二进制文件 application/octet-stream;
application/pkcs12, application/vnd.mspowerpoint, application/xhtml+xml,
application/xml xml 文件；
application/pdf pdf 文件；
aplication/json json 文件

text 类型，默认 text/plain，application 类型，默认 application/octet-stream
